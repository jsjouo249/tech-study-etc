OLTP - Online Transaction Processing
사용자의 PC에서 발생되는 트랜잭션을 실시간으로 DB의 데이터를 갱신하거나 조회하는 단위 작업을 처리하는 방식. - 실시간 처리

OLAP - Online Analytical Processing
DW를 활용하여 이미 저장되어 있는 데이터를 분석하고 사용자에게 유의미한 정보를 제공하는 처리방법.

OLAP Cube
데이터 구조로 매우 빠른 데이터 분석을 위해 최적화된 상태.

OLTP			OLAP
데이터 처리		데이터 분석
실시간 처리		수분 이내의 시간
테이블			분석된 정보
트랜잭션 중심의 데이터	수집된 정보 중심의 데이터
